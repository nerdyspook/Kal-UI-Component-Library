Kal-UI Color Scheme

Kal-UI comes with pre-defined colors, which are consistent in the overall design. The colors used are chosen on a generic basis, which can be accessed by pre-defined classes.

Default Palette

Below is the color palette, used by Kal-UI, listed with the hex-codes.

Default #0F172A
Primary #0EA5E9
Secondary #CBD5E1
Failure #DC2626
Success #22C55E
Warning #FB923C
